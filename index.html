Introduction

This course is to focus on data structure of programming. Data structure is a data structure is a data organization, management, and storage format that enables efficient access and modification.More precisely, a data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to the data. In this class, we will take a close look at various data structures, operations relating individual data structrues, programming techniques using data structures.
